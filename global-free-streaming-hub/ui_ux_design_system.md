## Design System for Global Free Streaming Hub

### Target: Global Free Streaming Hub
### Recommended Design System

PATTERN: Hero-Centric + Social Proof
Conversion: Emotion-driven with trust elements
CTA: Above fold, repeated after testimonials
Sections:
1. Hero
2. Features
3. Testimonials
4. Content Library
5. Pricing
6. Contact

STYLE: Dark Mode (OLED) + Glassmorphism
Keywords: Dark backgrounds, glass effects, premium feel, cinematic
Best For: Streaming platforms, entertainment, premium services
Performance: Excellent | Accessibility: WCAG AA

COLORS:
Primary:    #0a0a1a (Deep Space)
Secondary:  #1a1a2e (Midnight Blue)
CTA:        #00cc99 (Electric Green)
Background: #000000 (Pure Black)
Text:       #ffffff (Pure White)
Accent:     #00aaff (Electric Blue)
Notes: Cinematic dark palette with neon accents

Mood: Modern, cinematic, sophisticated
Best For: Streaming platforms, entertainment, tech
Google Fonts: https://fonts.google.com/share?selection.family=Inter:wght@400;500;600;700|Roboto+Slab:wght@400;500;600;700

KEY EFFECTS:
Glassmorphism cards + Smooth transitions (200-300ms) + Parallax scrolling + Hover states with glow effects

AVOID (Anti-patterns):
Bright backgrounds + Harsh animations + Light mode + AI purple/pink gradients

[ ] No emojis as icons (use SVG: Heroicons/Lucide)

[ ] Hover states with smooth transitions (150-300ms)
[ ] Light mode: text contrast 4.5:1 minimum
[ ] Focus states visible for keyboard nav
[ ] prefers-reduced-motion respected
[ ] Responsive: 375px, 768px, 1024px, 1440px
